About IoT_Health

Smart, portable, and secure system for monitoring vital signs.

What is IoT_Health?

IoT_Health is a complete, cost-effective system that monitors in real time body temperature, blood oxygen saturation (SpO₂), and ECG signals. It is based on a Raspberry Pi 5, records data in a local database, and displays them through a responsive web app. The architecture emphasizes security (AES-128-GCM encryption), portability (powerbank ~11.5 hours), and usability. It also integrates an embedded AI LLM enabling natural language queries directly on-device while ensuring privacy and responsiveness.

Key Features

Sensors

MCP9808 (Temperature), MAX30102 (SpO₂), AD8232 (ECG).

Data Security

AES-128-GCM encryption and local storage for privacy.

Web App

Responsive UI for mobile/desktop with easy monitoring of measurements.

Portability

Operates with a powerbank for ~11.5 hours.

AI Support

Natural language queries on the local database.

Scalability

Modular design for adding more sensors or cloud sync in the future.

Why It Matters

IoT_Health aims to bring health monitoring technologies closer to the user with low cost and high reliability. It can support scenarios of telemedicine, home care, and primary healthcare, especially in resource-limited environments.
